# Code of Conduct

This project adopts the **Contributor Covenant 2.1** as its code of conduct,
unchanged. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

Participation in this project — submitting PRs, opening issues, joining
discussions, or any other contribution — implies agreement with that document.

## Reporting concerns

Concerns about the conduct of a contributor or maintainer can be reported to:

- **Email**: opensource@zethic.com
- **Private security advisory**: <https://github.com/ZethicTech/react-native-update-gate/security/advisories/new>
  (use this for sensitive reports you'd like a written audit trail of)

Reports are reviewed by the project maintainers. We aim to respond within 7 days
and treat all reports confidentially.
